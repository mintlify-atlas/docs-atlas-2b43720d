---
title: "No-filtering DNS providers for pure resolution"
description: "DNS providers that return unmodified answers with no ad-blocking, malware filtering, or content restrictions — ideal for full DNS transparency."
---

If you want your device to receive the exact DNS response that the authoritative server returns, choose a no-filtering provider. These profiles set `censorship: false` in the source configuration, meaning the resolver never withholds or rewrites records for any domain. This is useful when you want to control filtering yourself, trust the upstream resolver entirely, or need access to domains that other providers might block.
